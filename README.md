# HeartDisease_Prediction
This is a streamlit based web app that leverages a Machine Learning Model trained on dataset to predict if there's any problem with the heart.

About the model:
The dataset was trained on the following models:
1. Logistic Regression
2. KNN
3. SVM
4. Decision Trees
5. Naive Bayes

Out of these, Logistic Regression provided the best results. However, keep in mind right now in the first version of the project, the hyperparameters of the models were not changed much, hence could be the lesser accuracy and f1-scores of some models.

The Accuracy and F1-Score of the Logistic Regression Model is:
logistic_regression - Accuracy: 0.8695652173913043, F1 Score: 0.8846153846153846
